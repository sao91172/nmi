How it Works

With the use of a Vue component, I can customize the breed of cat that each instance of the component shows a slideshow for. It takes in a 'prop' which gives the component the name that is used to match the correct id in the Cat API. The template contains the HTML that will be inserted when the component is used. Each data is collected for the objects and also returns each of the variables when the component is created. The component also calls loadNextImage with the prop. loadNextImage pre-loads the first 10 images for each breed to display them in the slideshow. The button created in the template is linked to the slideshow function, so each time it is clicked, the slideshow function is executed, and it displays the 10 pre-loaded images. The bar charts are created by making several colored divs that are sized according to their values.
